```bash

```

**Check CI-02 — test job has no if filter:**
```bash
python3 -c "import yaml; wf=yaml.safe_load(open('.github/workflows/ci.yml')); assert 'if' not in wf['jobs']['test']; print('CI-02 PASS')"
```

**Check CI-03 — build-and-push needs test:**
```bash
python3 -c "import yaml; wf=yaml.safe_load(open('.github/workflows/ci.yml')); needs=wf['jobs']['build-and-push']['needs']; assert needs=='test' or 'test' in needs; print('CI-03 PASS')"
```

**Check CI-04 — build-and-push has push condition:**
```bash
python3 -c "import yaml; wf=yaml.safe_load(open('.github/workflows/ci.yml')); assert 'push' in wf['jobs']['build-and-push']['if']; print('CI-04 PASS')"
```

**Check CI-05 — correct image name:**
```bash
grep -q 'ghcr.io/nguyenbuitk/claude-demo' .github/workflows/ci.yml && echo "CI-05 PASS"
```

**Check CI-06 — SHA + latest tags:**
```bash
python3 -c "
import yaml
wf=yaml.safe_load(open('.github/workflows/ci.yml'))

assert 'type=sha' in meta['with']['tags']
assert 'latest=auto' in meta['with']['flavor']
print('CI-06 PASS')
"
```

**Check CI-07 — packages: write permission:**
